# Week1

## 教学目标

* 理解 Linux 系统的基本使用方式
* 掌握通过 SSH 远程登录服务器的方法
* 掌握常用文件管理命令（查看、复制、目录操作）
* 能够在 Linux 环境下运行简单程序
* 具备基础的代码上传、项目拉取与运行能力
* 熟悉基本的文件传输工具使用

---

## 教学内容

### 1. 远程连接与环境准备

* 使用 Windows 终端（cmd / PowerShell）通过 SSH 连接 Ubuntu 服务器
* 登录流程与用户身份认证
* 修改用户密码并重新登录

### 2. Linux 文件系统与基本命令

* 目录结构（/、home 等）
* 文件查看（cat / less / head）
* 目录操作（cd / ls / pwd）
* 文件操作（cp / mv / mkdir）

### 3. 文件查找与操作

* 在根目录查找指定文件
* 文件复制到指定目录
* 创建个人工作目录（如 study）

### 4. 程序运行基础

* 上传本地代码到服务器
* 使用编译器或解释器运行程序（C/C++/Python）

### 5. 代码仓库使用

* 使用 git 克隆远程仓库（GitHub / Gitee）
* 基本部署与运行流程

### 6. 文件传输工具

* 使用 WinSCP 进行本地与服务器之间的文件互传
* 理解 SFTP 方式

### 7. 拓展实践

* 尝试部署开源项目（如 openclaw）
* 浏览并尝试终端趣味工具

---

## 验收内容

* 完成 SSH 登录并成功修改密码
* 在根目录找到 `helloworld.txt` 并复制到 `~/study` 目录
* 成功上传并运行一个自写程序（C/C++/Python 均可）
* 成功克隆一个 GitHub 或 Gitee 项目并运行
* 使用 WinSCP 完成至少一次文件上传与下载
* 安装 `pokemonsay` 项目并完成自定义图片配置并运行
